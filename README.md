# Go-Eureka

Simple implementation of Spring Eureka Server in Gin

## Installation

```bash
go get -u github.com/Nguyen-Hoang-Nam/go-eureka
```

## Usage

```go
package main

import "github.com/Nguyen-Hoang-Nam/go-eureka/eureka"

func main() {
    eureka.Run("9000")
}
```
